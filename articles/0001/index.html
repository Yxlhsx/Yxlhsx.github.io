Centos7 yum 安装 JDK1.8

检查当前系统是否安装 jdk

rpm -qa | grep java

如果没有输入信息表示没有安装，反之若有输出内容，则执行**rpm -e --nodeps ???**进行卸载，问号为输出的信息名。

例如：

rpm -e --nodeps java-1.8.0-openjdk-1.8.0.232.b09-0.el7_7.x86_64

查看 yum 源的 java 包

yum list java*

安装 jdk1.8

yum -y install java-1.8.0-openjdk

等待一会儿。

检查是否安装成功

java -version

显示版本则安装成功，这样的安装方式可以省去手动配置环境变量环节。
