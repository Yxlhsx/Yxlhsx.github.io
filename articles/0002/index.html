下载 nginx

进入官网下载 nginx。 点这里进入官网下载界面 可以看到很多版本，我们这里选择较新的稳定版。 下载完成后，将下载好的压缩包移动到 /usr/local下，然后解压。

tar -zxvf nginx-1.18.0.tar.gz

安装 nginx

解压后切换到安装目录下。

cd /usr/local/nginx/nginx-1.18.0

然后执行安装：

./configure --prefix=/usr/local/nginx   #配置安装目录

make

make install

检测是否安装成功

进入到上面配置的安装目录

cd /usr/loca/nginx/sbin

./nginx -t
